# MiniApp

React Native (Expo SDK 55) post viewer with JWT auth.

## Setup

```bash
npm install
```

## Run

```bash
npx expo start        # Dev server (press i = iOS sim, a = Android emu)
npx expo start --ios  # iOS Simulator directly
npx expo start --android  # Android Emulator directly
```

Or install [Expo Go](https://expo.dev/go) on your phone and scan the QR code.

## Other Commands

```bash
npm run generate-api  # Regenerate API client from OpenAPI spec
npx tsc --noEmit      # Type check
npm test              # Run tests
```
